# Manage your expenses

"Manage your expenses" is personal expenses management web client application that allows you to track how much money have you spent.

## Technologies

* React 16.13.1
* React-Redux 7.2.0
* Redux 4.0.5
* Redux-form 8.3.6
* Redux-thunk 2.3.0
* Node-sass 4.14.1
* Axios 0.19.2
